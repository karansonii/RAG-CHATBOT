# RAG-CHATBOT
Streamlit-based multilingual (EN/SW) AI chatbot with RAG for DOCX/PDF/PPTX. Uses OCR, sentence-transformers + FAISS retrieval, and Google Gemini for answers. Supports voice I/O (STT + gTTS), Mermaid flowcharts, chat history, and .env config. Interactive, document-aware assistant with prompt safeguards.
